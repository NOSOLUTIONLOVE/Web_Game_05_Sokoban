## 四、风险与回滚

| 风险 | 检测 | 回滚策略 |
| --- | --- | --- |
| 撤销栈内存膨胀 | 内存监控 | MAX_HISTORY=200 |
| 死锁状态无法挽救 | 试玩测试 | V2 死锁检测 + 提示 |
| 触屏滑动误触 | 真机测试 | 阈值 30px + 防抖 |
| 关卡难度失衡 | 试玩 | 参考 Microban 调整 |
| Canvas 性能问题 | DevTools FPS | 减少 drawCell 内的循环 |
| 状态机非法操作 | 单元测试 | GameEngine 状态守卫 100% 覆盖 |
| React 重渲染过多 | Profiler | 引擎内部状态私有，仅事件触发 store 更新 |
| 移动端 ActionBar 遮挡 | 真机测试 | 透明背景 + 底部固定 + 安全区适配 |
