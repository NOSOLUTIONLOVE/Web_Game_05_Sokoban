## 二、MVP 验收标准

### F-01：7 种 CellType 渲染

- ✅ `wall` 深灰带砖纹高光
- ✅ `floor` 木地板带斜线纹理
- ✅ `target` 黄色十字（中心点 + 4 角方块），缓慢脉冲
- ✅ `box` 棕色木箱 + 板条横纹
- ✅ `box_on_target` 绿色木箱 + 持续微脉冲
- ✅ `player` 蓝色圆形身 + 头 + 4 方向姿态
- ✅ `player_on_target` 同 player，脚下有黄色目标点标记

### F-02：移动控制

| 输入 | 行为 |
| --- | --- |
| ↑ / W | 向上移动 |
| ↓ / S | 向下移动 |
| ← / A | 向左移动 |
| → / D | 向右移动 |
| 触屏滑动 | 滑动方向 ≥ 30px → 对应方向移动 |

### F-03：推箱 + 撞墙检测

- 移动到墙 → 推不动（触发 blocked 动画）
- 移动到箱子 + 箱子后是墙 → 推不动
- 移动到箱子 + 箱子后是另一箱子 → 推不动
- 移动到箱子 + 箱子后是空地 → 推箱成功
- 推箱成功 + 箱子归位 → 触发 settle 动画

### F-04：胜利判定

- 所有箱子都在目标点上 → 胜利
- 触发 `onWin` 回调
- 自动持久化最佳步数 + 最佳推箱数
- 解锁下一关

### F-05：内置 20 关

- 来自 Microban XS（仓库风格）改编
- 难度从 1（简单）到 5（困难）递增

### F-06：撤销（U 键）

- 200 步历史栈
- U 键触发 `engine.undo()`
- 撤销后移动 / 推箱计数 -1

### F-07：重置（R 键）

- R 键触发 `engine.resetLevel()`
- 移动 / 推箱计数清零
- 撤销栈清空
- 阶段回到 playing

### F-08：暂停 / 继续

- P 键或 Esc 键 → 暂停
- 暂停时显示遮罩
- 再按一次 → 继续

### F-09：localStorage 持久化

- 键名：`sokoban:store`（Zustand persist）+ `sokoban:progress`（关卡进度）
- 持久化字段：unlockedLevels / levelBestMoves / levelBestPushes / levelCleared / audioEnabled

### F-10：移动端 ≤ 375px

- Canvas 自适应缩放
- ActionBar（仅 md 以下显示）提供撤销 / 重做 / 重置
- 触屏滑动可玩

### F-20：Vercel 部署

- `vercel.json` 配置 base `./`
- 部署后可在 Vercel 域名访问

## 三、V2 验收标准

### F-11：重做（Y 键）

- 撤销后可重做
- 重新执行被撤销的移动

### F-12：死锁检测

- **角落死锁**：箱子被推到墙 + 目标点对角
- **边墙死锁**：箱子被推到墙边，垂直方向是墙
- 触发后弹出红色 toast（3 秒后消失）

### F-13：关卡选择

- Dialog 模式
- 显示 20 关网格（4-5 列）
- 已通关关卡绿色背景
- 当前关卡高亮
- 未解锁关卡灰色 + 锁图标

### F-14：关卡解锁

- 通关后自动解锁下一关
- localStorage 持久化

### F-15：音效

| 事件 | 音效 |
| --- | --- |
| 普通移动 | 短促轻响（300Hz，50ms） |
| 推箱 | 低沉推力（150Hz，100ms） |
| 撤销 | 反向音（400→200Hz，80ms） |
| 推不动 | 撞击 thud（80Hz，150ms） |
| 胜利 | 上行音阶（C-E-G-C，500ms） |

### F-16：触屏滑动

- 滑动方向 ≥ 30px → 对应方向移动
- 滑动阈值防误触
- canvas `touch-action: none` 防止滚动穿透

### F-17：归位绿光

- 推箱成功且箱子归位 → 绿光闪烁 2 次（400ms）

### F-18：推不动抖动

- 推不动 → 工人抖动 200ms（4 个周期）

### F-19：ActionBar

- 移动端底部
- 撤销 / 重做 / 重置 三个按钮
- 仅 md 以下显示
